# ImageCheck
图片或者头像选择组件，提供了从相机或者相册选取图片进行剪切的功能，剪切后会得到图片的Bitmap，可用于上传到服务器或者本地使用


使用方法很简单，只需要调用ImageCheck类的show()方法即可,
